Navigation area

This area allows you to navigate the folders and groups tree. Its width is adjustable and it can be hidden clicking on the button
To recover the navigation area, just click on the button
The tree does not display members of groups (sub-groups or individuals).

You can deploy or undeploy a branch of the tree clicking the buttons and beside the folders names.
An overview on an icon associated whith a folder () or a group () lets you know what your rights on this folder or on this group are.
Right click on the name of a folder/group get a contextual menu that provides actions allowed on this folder/group.
  • Create a folder : create a folder in the selected folder
  • Create a group : create a group in the selected folder
  • Copier le groupe : copy the group previously cut in the selected folder.
  • Search : search a subject from the selected folder or from the parent folder of the selected group
  • Delete : delete the selected folder/group
  • Admin : admin the selected folder/group
  • Properties : display the properties of the selected folder/group
  • Join : join the selected group
  • Unjoin : unjoin the selected group
  • Couper le groupe : Cut the group from the current folder to copy it elsewhere
  • Copy members : copy the members of the selected group to paste them in another group
  • Paste members : paste the members copied earlier in the selected group
The action Cut/copy group can be made by grag'n'drop.
