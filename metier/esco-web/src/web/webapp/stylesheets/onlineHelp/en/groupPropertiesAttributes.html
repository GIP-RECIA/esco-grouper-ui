Group properties : Attributes

Dans cet onglet, ESCO-Grouper affiche les attributs du groupe :
  • Extension : nom interne à Grouper du groupe ;
  • Extension affichée : nom du groupe qui sera affiché, notamment dans l'arborescence ;
  • Description : description du groupe ;
  • UUID : numéro identifiant le groupe dans Grouper.
En fonction du type de groupe, des informations complémentaires sont affichées sous la rubrique "Types personnalisés"

En fonction des droits de l'utilisateur connecté, un certain nombre de boutons peuvent s'afficher au bas de la page alors que l'onglet " Attributs" est sélectionné :
  • Administrer : permet d'administrer le groupe ;
  • Supprimer : supprime le groupe et se place sur le dossier juste au dessus dans l'arborescence ;
