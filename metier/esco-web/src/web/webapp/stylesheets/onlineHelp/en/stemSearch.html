Folder search

Cette page permet de rechercher un dossier dans une arborescence.
L'utilisateur précise les éléments nécessaires à la recherche et clique sur le bouton " Recherche" pour lancer la recherche.
Le résultat de la recherche s'affiche dans un tableau. L'utilisateur peut alors choisir le(s) dossier(s) qu'il souhaite grâce à(aux) case(s) à cocher, avant de cliquer sur le bouton " Ajouter" pour ramener sa sélection vers l'opération en cours.
Si l'utilisateur clique sur le bouton " Retour", il revient à l'opération en cours sans ramener un résultat de recherche.

Il y a différents éléments qui sont nécessaires à la recherche et que l'utilisateur aura à préciser pour obtenir le résultat souhaité.
Terme de recherche :
Dans cette zone, l'utilisateur entre une chaine de recherche. L'utilisateur peut utiliser le caractère "*" pour remplacer tout caractère. Par exemple, pour rechercher tous les dossiers dont l'extension commence par "Etab", l'utilisateur entrera "Etab*" dans la zone "Terme de recherche".

A partir de :
Ici, l'utilisateur sélectionne le dossier à partir duquel s'effectuera la recherche. Par défaut, le dossier sélectionné est le dossier courant, c'est à dire celui à partir duquel l'utilsateur a demandé la recherche. Si l'utilisateur souhaite effectuer la recherche sur toute l'arborescence, il cliquera sur le symbole .

Critère de recherche" :
Cette zone permet de préciser sur quelle partie du nom du dossier le terme de recherche sera recherché.
Un dossier possède un Nom et un Nom affiché.
Le Nom d'un dossier est le nom complet par lequel il est enregistré dans le gestionnaire de groupes Grouper. Le Nom affiché est le nom complet du dossier tel qu'il apparaît dans l'arborescence de ESCO-Grouper.
L'Extention affichée d'un dossier est le dernier élément de son Nom affiché.
Prenons l'exemple du dossier qui a pour Extension affichée "Administration locale du cahier de texte". Le nom de ce dossier est "esco:admin:CAHIER_TEXTE_CAEN:local" et son Nom affiché est "ESCO:Admin:Cahier de texte:Administration locale du cahier de texte".
