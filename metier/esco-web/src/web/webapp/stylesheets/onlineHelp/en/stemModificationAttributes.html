Folder edition : Attributes

Dans cet onglet, l'utilisateur définit ou modifie les attributs du dossier. Trois informations sont nécessaires :
  • Extension : nom interne à Grouper du dossier ;
  • Extension affichée : nom du dossier qui sera affiché, notamment dans l'arborescence ;
  • Description : description du dossier, c'est un texte libre.
Sur un dossier, ces informations ne peuvent être modifiées que si l'utilisateur connecté possède les droits suffisants.
