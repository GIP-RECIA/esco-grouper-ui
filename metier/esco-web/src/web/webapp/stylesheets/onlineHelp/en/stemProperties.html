Folder properties

Cette page affiche les propriétés du dossier sélectionné :
  • Extension : nom interne à Grouper du dossier ;
  • Extension affichée : nom usuel du dossier notamment tel qu'il apparaît dans l'arborescence des dossiers ;
  • Description : Description du dossier ;
  • uuid : numéro interne du dossier dans Grouper.
En fonction des droits de l'utilisateur connecté, un certain nombre de boutons peuvent s'afficher au bas de la page :
  • Administrer : permet d'administrer le dossier ;
  • Supprimer : supprime le dossier et se place sur le dossier juste au dessus dans l'arborescence ;
  • Créer un dossier : permet de créer un nouveau dossier dans le dossier ;
  • Créer un groupe : permet de créer un nouveau groupe dans le dossier.
