Groupe edition: Privileges

Cet onglet permet de voir, modifier, ajouter ou supprimer des privilèges sur le groupe. Il existe 6 privilèges possibles :
  • A : droit d'administrer le groupe ;
  • M : droit de modifier le groupe, c'est à dire le droit d'ajouter ou de supprimer des membres ;
  • L : droit de lire le groupe, c'est à dire de consulter la liste des membres ;
  • V : droit de voir le groupe dans l'arborescence et d'afficher ses propriétés, mais sans pouvoir lister ses membres ;
  • + : droit de s'abonner au groupe ;
  • - : droit de se désabonner du groupe ;
Les droits sont cumulatifs, c'est à dire, par exemple, qu'un sujet qui a le droit de modifier le groupe aura également le droit de lire et de voir le groupe. Ou encore qu'un sujet qui a le droit de s'abonner au groupe aura le droit de voir ce groupe.
L'utilisateur qui crée le groupe aura par défaut le droit d'administrer ce groupe.

Une liste de choix permet d'accéder aux :
  • Privilèves directs : privilèges que possèdent directement des sujets (individus ou groupes) sur le groupe ;
  • Privilèges indirects : privilèges que possèdent indirectement des sujets sur le groupe (parce qu'ils font partie d'un groupe qui a des privilèges sur le groupe) ;
  • Privilèges directs et indirects : privilèges directs et indirects que possèdent des sujets sur le groupe.
Lorsque l'utilisateur sélectionne les privilèges directs, il peut modifier, ajouter ou supprimer des privilèges.

Pour modifier les privilèges d'un sujet sur le groupe, il suffit de cliquer sur les cases à cocher correspondantes dans le tableau.

Pour ajouter des privilèges à un nouveau sujet, il faut cliquer sur " Ajout" et rechercher le sujet concerné. Une fois la recherche effectuée ESCO-Grouper revient sur l'onglet " Privilèges" et le nouveau sujet est affiché en vert dans la liste. Il suffit ensuite de cocher les privilèges souhaités pour ce sujet.

Pour supprimer tous les privilèges d'un sujet sur le groupe, il faut sélectionner ce sujet dans le tableau et cliquer sur " Supprimer".

Le bouton " Enregistrer" permet d'accéder à la page de récapitulation des opérations effectuées. Une validation de cette page enregistre la création ou les modifications.
Le bouton " Annuler" permet d'annuler les opérations en cours et de revenir sur la page des " Attributs", avec les informations précédemment saisies.

Si l'utilisateur quitte l'édition du groupe (par exemple en cliquant sur un autre groupe dans la zone de navigation), un message lui propose d'enregistrer les modifications ou de les annuler.
