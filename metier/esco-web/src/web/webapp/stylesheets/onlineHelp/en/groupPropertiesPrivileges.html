Group properties : Privileges

Dans cet onglet, ESCO-Grouper affiche les privilèges que possède le groupe sur des dossiers ou sur d'autres groupes. Le fait de voir les privilèges sur les dossiers ou les groupes s'effectue grâce à une des deux listes de choix.

L'autre liste de choix permet d'affcher :
  • les Privilèges directs : privilèges attribués directement au groupe sur des dossiers ou des groupes ;
  • les Privilèges indirects : privilèges attribués indirectement au groupe (parce que ce groupe est membre d'un autre groupe qui possède des privilèges) sur des dossiers ou des groupes ;
  • les privilèges directs et indirects : privilèges attribués directement ou indirectement au groupe sur des dossiers ou des groupes.
Le tableau affiche donc la liste des dossiers ou des groupes sur lesquels le groupe a des privilèges.

Les privilèges possibles sur les dossiers sont :
  • CD : droit de créer un dossier dans le dossier ;
  • CG : drout de créer un groupe dans le dossier.
Les privilèges possibles sur les groupes sont :
  • A : droit d'administrer le groupe ;
  • M : droit de modifier le groupe, c'est à dire le droit d'ajouter ou de supprimer des membres ;
  • L : droit de lire le groupe, c'est à dire de consulter la liste des membres ;
  • V : droit de voir le groupe dans l'arborescence et d'afficher ses propriétés, mais sans pouvoir lister ses membres ;
  • + : droit de s'abonner au groupe ;
  • - : droit de se désabonner du groupe ;
En fonction des droits de l'utilisateur connecté, et lorsque Privilèges directs est choisi, il est possible d'ajouter ou de supprimer des privilèges au groupe.

Dans la liste des dossiers/groupes affichés, il est possible d'ajouter ou de retirer des privilèges en cochant ou décochant les cases dans les colonnes des droits.

Pour ajouter des privilèges au groupe sur un nouveau dossier/groupe, il faut cliquer sur " Ajout" et rechercher le dossier/groupe concerné. Une fois la recherche effectuée ESCO-Grouper revient sur l'onglet " Privilèges" et le nouveau dossier/groupe est affiché en vert dans la liste. Il suffit ensuite de cocher les privilèges souhaités pour le groupe.

Pour supprimer tous les privilèges du groupe sur un dossier/groupe, il faut sélectionner ce dossier/groupe dans la liste et cliquer sur " Supprimer".
