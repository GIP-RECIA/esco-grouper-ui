Group properties

Cette page affiche les propriétés du groupe sélectionné. En fonction des droits de l'utilisateur connecté, jusqu'à 4 onglets peuvent apparaître :
  • Attributs : attributs du groupe ;
  • Membres : membres du groupe ;
  • Appartenances : groupes dont est membre le groupe sélectionné ;
  • Privilèges : privilèges du groupe sur d'autres groupes ou sur des dossiers.
En fonction des droits de l'utilisateur connecté, un certain nombre de boutons peuvent s'afficher au bas de la page alors que l'onglet " Attributs" est sélectionné :
  • Administrer : permet d'administrer le groupe ;
  • Supprimer : supprime le groupe et se place sur le dossier juste au dessus dans l'arborescence ;
