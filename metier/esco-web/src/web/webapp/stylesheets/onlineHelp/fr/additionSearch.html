Recherche pour ajout

Cette page permet de rechercher des sujets, c'est à dire des individus ou des groupes pour les ajouter au tableau associé à une opération en cours : ajout de membres à un groupe, ajout de provilèges...

L'utilisateur précise les éléments nécessaires à la recherche et clique sur le bouton " Recherche" pour lancer la recherche.
Le résultat de la recherche s'affiche dans un tableau.
L'utilisateur peut alors choisir le(s) sujet(s) qu'il souhaite grâce à(aux) case(s) à cocher, avant de cliquer sur le bouton " Ajouter" pour ramener sa sélection vers l'opération en cours.
Si l'utilisateur clique sur le bouton " Retour", il revient à l'opération en cours sans ramener un résultat de recherche.

Pour rechercher un individu, l'utilisateur indique le nom, éventuellement suivi d'un blanc et du prénom, de l'individu recherché dans la zone "Terme de recherche". L'utilisateur peut utiliser le caractère "*" pour remplacer tout caractère. Par exemple, pour rechercher tous les individus dont le nom commence par "Tar", l'utilisateur entrera "Tar*" dans la zone "Terme de recherche".
La recherche d'un individu n'est disponible que si cela a un sens pour l'opération en cours. Par exemple, si l'utilisateur attribue des privilèges dans l'édition d'un individu, ces privilèges ne peuvent être placés que sur un groupe ; la recherche d'individu est donc désactivée.

Pour rechercher un groupe, plusieurs éléments seront à préciser.
Terme de recherche :
Dans cette zone, l'utilisateur entre une chaine de recherche. L'utilisateur peut utiliser le caractère "*" pour remplacer tout caractère. Par exemple, pour rechercher tous les groupes dont l'extension commence par "Prof", l'utilisateur entrera "Prof*" dans la zone "Terme de recherche".
A partir de :
Ici, l'utilisateur sélectionne le dossier à partir duquel s'effectuera la recherche. Par défaut, le dossier sélectionné est le dossier courant, c'est à dire celui à partir duquel l'utilisateur a lancé l'opération. Si l'utilisateur souhaite effectuer la recherche sur toute l'arborescence, il cliquera sur le symbole .
Critère de recherche :
Cette zone permet de préciser sur quelle partie du nom du groupe le terme de recherche sera recherché.
Un groupe possède un Nom et un Nom affiché.
Le Nom d'un groupe est le nom complet par lequel il est enregistré dans le gestionnaire de groupes Grouper. Le Nom affiché est le nom complet du groupe tel qu'il apparaît dans l'arborescence de ESCO-Grouper.
L'Extention affichée d'un groupe est le dernier élément de son Nom affiché.
Prenons l'exemple d'un groupe qui a pour Extension affichée "CLAUDE DE FRANCE" et qui est contenu dans le dossier "ESCO:Admin:Cahier de texte:Administration locale du cahier de texte". Le nom de ce groupe est "esco:admin:CAHIER_TEXTE_CAEN:local:CLAUDE DE FRANCE_0410017W" et son Nom affiché est "ESCO:Admin:Cahier de texte:Administration locale du cahier de texte:CLAUDE DE FRANCE".

Pour rechercher des personnes de groupes, il faut commencer par rechercher le ou les groupes exactement comme indiqué plus haut. Mais lorsque l'utilisateur clique sur " Recherche", ESCO-Grouper affiche une fenêtre avec la liste des groupes répondant au critère de recherche. L'utilisateur choisit alors le ou les groupes grâce aux cases à cocher, et clique sur le bouton " Rechercher membres" pour que la liste des personnes qui sont membres directs du ou des groupes sélectionnés s'affiche dans le formulaire de recherche.
Attention : si un groupe n'a que d'autres groupes pour membres directs (aucune personne), alors rien ne s'affichera dans le formulaire de recherche excepté le message "Aucun résultat pour la recherche".
