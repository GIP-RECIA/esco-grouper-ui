Édition de dossier : Privilèges

Cet onglet permet de voir, modifier, ajouter ou supprimer des privilèges sur le dossier. Il existe deux privilèges possibles :
  • CD : droit de créer des dossiers dans le dossier ;
  • CG : droit de créer des groupes dans le dossier.
L'utilisateur qui crée le dossier possède par défaut le droit "CD" sur ce dossier. S'il désire avoir également le droit "CG" sur ce dossier, il faut explicitement le rajouter.

Une liste de choix permet d'accéder aux :
  • Privilèves directs : privilèges que possèdent directement des sujets (individus ou groupes) sur le dossier ;
  • Privilèges indirects : privilèges que possèdent indirectement des sujets sur le dossier (parce qu'ils font partie d'un groupe qui a des privilèges sur le dossier) ;
  • Privilèges directs et indirects : privilèges directs et indirects que possèdent des sujets sur le dossier.
Lorsque l'utilisateur sélectionne les privilèges directs et s'il possède les droits suffisants, il peut modifier, ajouter ou supprimer des privilèges.

Pour modifier les privilèges d'un sujet sur le dossier, il suffit de cliquer sur les cases à cocher correspondantes dans le tableau.

Pour ajouter des privilèges à un nouveau sujet, il faut cliquer sur " Ajout" et rechercher le sujet concerné. Une fois la recherche effectuée ESCO-Grouper revient sur l'onglet " Privilèges" et le nouveau sujet est affiché en vert dans la liste. Il suffit ensuite de cocher les privilèges souhaités pour ce sujet.

Pour supprimer tous les privilèges d'un sujet sur le dossier, il faut sélectionner ce sujet dans le tableau et cliquer sur " Supprimer".
