Édition de dossier

Cette page permet de créer ou de modifier un dossier. Deux onglets sont disponibles :
  • Attributs : permet de définir les attributs du dossier ;
  • Privilèges : permet de définir les privilèges sur le dossier.
Le bouton " Enregistrer" permet d'accéder à la page de récapitulation des modifications effectuées. Une validation de cette page enregistre les modifications.
Il n'est pas nécessaire d'enregistrer une à une les modifications effectuées sur le dossier. Celles-ci sont historisées jusqu'à l'enregistrement effectif. Si l'utilisateur quitte l'édition du dossier (par exemple en cliquant sur un autre dossier dans la zone de navigation), un message lui propose d'enregistrer les modifications ou de les annuler.

Le bouton " Annuler" permet d'annuler les opérations en cours.
