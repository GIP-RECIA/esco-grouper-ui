Propriétés d'un individu

Cette page sert de page d'accueil lors de la connexion à ESCO-Grouper en affichant les propriétés de la personne connectée.
Mais cette page affiche les propriétés de n'importe quel individu, par exemple lors d'un clic sur son nom dans une liste (liste des membres d'un groupe, résultat d'une recherche de personnes...).
Cette page affiche jusqu'à 4 onglets, en fonction des droits de l'utilisateur connecté :
  • Attributs : attributs de l'individu, issus de l'annuaire LDAP ;
  • Appartenances : groupes auxquels appartient l'individu ;
  • Privilèges : privilèges que cet individu possède sur des groupes et des dossiers ;
  • Abonnements : groupes auxquels l'individu est abonné ou susceptible d'être abonné.
En fonction des droits de l'utilisateur, il est possible de modifier les appartenances, les privilèges ou les abonnements de l'individu.
Une fois les modifications effectuées, il faut les enregistrer. Pour cela il suffit de cliquer sur le bouton " Enregistrer". La page de récapitulation de toutes les modification effectuées pour l'individu est alors affichée. Une validation de cette page enregistre les modifications.
Il n'est pas nécessaire d'enregistrer une à une les modifications effectuées. Celles-ci sont historisées jusqu'à l'enregistrement effectif. Si l'utilisateur quitte l'édition des propriété de l'individu (par exemple en cliquant sur le nom d'un groupe dans la zone de navigation), un message lui propose d'enregistrer les modifications ou de les annuler.

Cliquer sur le bouton " Annuler" annule toutes les modifications en cours et affiche l'onglet " Attributs" de l'individu.

Il faut noter que l'affichage des onglets " Privilèges" et " Abonements" peut demander un certain temps à cause du nombre de groupes existant dans Grouper.
