Zone de naviguation

Cette zone permet de naviguer dans l'arborescence des dossiers et des groupes. Sa largeur est ajustable et elle peut être cachée en cliquant sur le bouton
Pour retrouver la zone de navigation, il suffit de cliquer sur le bouton
L'arborescence n'affiche pas les membres des groupes (sous-groupes ou individus).

Vous pouvez déplier ou replier une branche de l'arbre en cliquant sur les boutons et à côté des noms des dossiers.
Un survol sur une icône associée à un dossier () ou un groupe () vous permet de savoir quels sont vos droits sur ce dossier ou ce groupe.
Un clic droit sur le nom d'un dossier/groupe vous permet d'obtenir un menu contextuel qui propose les actions permises sur ce dossier/groupe :
  • Créer un dossier : créer un dossier dans le dossier sélectionné
  • Créer un groupe : créer un groupe dans le dossier sélectionné
  • Copier le groupe : copie le groupe précédemment coupé dans le dossier sélectionné
  • Rechercher : rechercher un individu ou un groupe, à partir du dossier sélectionné ou du dossier père du groupe sélectionné
  • Supprimer : supprimer le dossier/groupe sélectionné
  • Administrer : modifier les caractéristiques du dossier/groupe
  • Propriétés : afficher les propriétés d'un dossier/groupe
  • S'abonner : s'abonner au groupe sélectionné
  • Se désabonner : se désabonner du groupe sélectionné
  • Couper le groupe : Coupe le groupe du dossier dans lequel il est placé pour le copier ailleurs
  • Copier les membres : copier les membres du groupe sélectionné pour pouvoir les insérer dans autre
  • Coller les membres : coller les membres copiés antérieurement dans le groupe sélectionné
L'opération Couper/copier un groupe peut s'effectuer par glisser/déplacer dans l'arborescence.
Dans l'arborescence, vous ne pouvez voir que les groupes () pour lesquels vous avez au minimum le droit "voir".
Par contre, sur les dossiers (), il n'y a pas de droit "voir" (seuls les droits "Créer Dossier" et "Créer Groupe" peuvent être positionnés) ; les dossiers sont donc affichés même si vous n'avez pas de droit dessus : ce n'est pas parce que vous n'avez pas le droit de créer un groupe ou un dossier sous un dossier que vous ne pouvez pas voir un groupe de ce dossier.

Le bouton permet de revenir sur la page d'accueil (propriétés de l'utilisateur connecté).
Le bouton vous amène sur la page Recherche. De cette page vous pouvez rechercher un sujet (individu ou groupe).
