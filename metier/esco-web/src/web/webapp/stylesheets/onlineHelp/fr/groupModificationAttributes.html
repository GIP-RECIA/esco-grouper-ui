Édition de groupe : Attributs

Cet onglet permet de saisir ou de modifier les attributs d'un groupe. Ces attributs sont affichés sur 4 pavés :
  • Attributs du groupe : ce pavé présente les attributs de base nécessaires à la définition du groupe
    • Extension : nom interne à Grouper du groupe ;
    • Extension affichée : nom du groupe qui sera affiché, notamment dans l'arborescence ;
    • Description : description du groupe, c'est un texte libre.
  • Privilèges par défaut sur ce groupe : ce sont les privilèges que tout utilisateur aura sur le groupe. Par exemple, si vous cochez "Administration", alors tous les utilisateurs qui se connectent à ESCO-Grouper pourront administrer ce groupe.
  • Types personnalisés : permet de définir le type du groupe. Le type par défaut est le type "Groupe standard", qui est le seul type disponible actuellement.
  • Contextes d'utilisation : ce pavé est réservé à un usage ultérieur.
Jusqu'à 3 boutons sont disponibles lors de cette première étape de création/modification du groupe :
  • " Supprimer" : permet de supprimer le groupe, si celui-ci existe déjà ;
  • " Annuler" : annule les modifications en cours et réaffiche la page des attributs vide dans le cas d'une création de groupe ou avec les attributs initiaux dans le cas d'une modification de groupe ;
  • " Suivant" : permet de passer à la deuxième étape, celle de la définition des privilèges sur le groupe.
