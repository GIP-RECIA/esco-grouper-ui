Simple search

Cette page permet de rechercher un sujet, c'est à dire un individu ou un groupe.
L'utilisateur précise les éléments nécessaires à la recherche et clique sur le bouton " Recherche" pour lancer la recherche.
Le résultat de la recherche s'affiche dans un tableau.
L'utilisateur peut alors cliquer sur le nom d'un sujet pour accéder aux propriétés de ce sujet.

Pour rechercher un individu, l'utilisateur indique le nom, éventuellement suivi d'un blanc et du prénom, de l'individu recherché dans la zone "Terme de recherche". L'utilisateur peut utiliser le caractère "*" pour remplacer tout caractère. Par exemple, pour rechercher tous les individus dont le nom commence par "Tar", l'utilisateur entrera "Tar*" dans la zone "Terme de recherche".

Pour rechercher un groupe, plusieurs éléments seront à préciser.
Terme de recherche :
Dans cette zone, l'utilisateur entre une chaine de recherche. L'utilisateur peut utiliser le caractère "*" pour remplacer tout caractère. Par exemple, pour rechercher tous les groupes dont l'extension commence par "Prof", l'utilisateur entrera "Prof*" dans la zone "Terme de recherche".
A partir de :
Dans une recherche simple, toute l'arborescence est explorée. De fait, le symbole est affiché et ne peut ètre modifié.
Critère de recherche" :
Cette zone permet de préciser sur quelle partie du nom du groupe le terme de recherche sera recherché.
Un groupe possède un Nom et un Nom affiché.
Le Nom d'un groupe est le nom complet par lequel il est enregistré dans le gestionnaire de groupes Grouper. Le Nom affiché est le nom complet du groupe tel qu'il apparaît dans l'arborescence de ESCO-Grouper.
L'Extention affichée d'un groupe est le dernier élément de son Nom affiché.
Prenons l'exemple d'un groupe qui a pour Extension affichée "CLAUDE DE FRANCE" et qui est contenu dans le dossier "ESCO:Admin:Cahier de texte:Administration locale du cahier de texte". Le nom de ce groupe est "esco:admin:CAHIER_TEXTE_CAEN:local:CLAUDE DE FRANCE_0410017W" et son Nom affiché est "ESCO:Admin:Cahier de texte:Administration locale du cahier de texte:CLAUDE DE FRANCE".
