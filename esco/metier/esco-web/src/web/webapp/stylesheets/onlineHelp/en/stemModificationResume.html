Folder edition : Resume

Cette page affiche un résumé des modifications effectuées.
Pour que ces modifications soient prises en compte, il faut cliquer sur le bouton " Enregistrer".
Si des erreurs apparaissent lors de l'enregistrement (droits insuffisants par exemple), la page récapitulative se réaffiche et un message d'erreur informe l'utilisateur. Une fois les erreurs corrigées, l'enregistrement peut s'effectuer.

Le bouton " Annuler" permet de revenir sur l'édition du dossier. Les informations saisies précédemment lors de l'édition ne sont pas perdues.
