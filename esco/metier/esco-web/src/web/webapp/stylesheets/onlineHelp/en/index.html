ACCEUIL

Dans cette documentation, vous trouverez tout ce qui est nécessaire à l'utilisation de ESCO-Grouper.
Les différentes pages d'aide sont accessible au cours de l'utilisation de ESCO-Grouper en cliquant sur les symboles .
Les documentations concernent :
  • 1 - la zone de navigation
  • 2 - les propriétés d'un individu, avec :
    • 2.1 - les attributs d'un individu
    • 2.2 - les appartenances d'un individu
    • 2.3 - les privilèges d'un individu
    • 2.4 - les abonnements d'un individu
    • 2.5 - le résumé des modifications sur un individu
  • 3 - les propriétés d'un groupe, avec :
    • 3.1 - les attributs d'un groupe
    • 3.2 - les membres d'un groupe
    • 3.3 - les appartenances d'un groupe
    • 3.4 - les privilèges d'un groupe
    • 3.5 - le résumé des modifications des propriétés d'un groupe
  • 4 - les propriétés d'un dossier
  • 5 - l'édition d'un groupe, avec :
    • 5.1 - les attributs d'un groupe
    • 5.2 - les privilèges sur un groupe
    • 5.3 - le résumé des modifications apportées à un groupe
  • 6 - l'édition d'un dossier, avec :
    • 6.1 - les attributs d'un dossier
    • 6.2 - les privilèges sur le dossier
    • 6.3 - le résumé de l'édition d'un dossier
  • 7 - les recherches, avec :
    • 7.1 - la recherche simple
    • 7.2 - la recherche pour ajout
    • 7.3 - la recherche de dossiers
