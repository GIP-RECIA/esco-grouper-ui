Propriétés de groupe : Appartenances

Dans cet onglet, ESCO-Grouper affiche tous les groupes dont le groupe est membre.
Une liste de choix permet de préciser les appartenaces que l'on souhaite afficher :
  • Appartenances directes : liste des groupes dont le groupe est membre direct
  • Appartenances indirectes : liste des groupes dont le groupe est membre indirect
  • Appartenances directes et indirectes : liste des groupes dont le groupe est membre direct ou indirect
En fonction des droits de l'utilisateur connecté, et lorsque Appartenances directes est choisi, il est possible d'ajouter ou de supprimer des appartenances.
Pour ajouter le groupe dans un autre groupe, il faut cliquer sur " Ajout" et rechercher le groupe concerné. Une fois la recherche efectuée ESCO-Grouper revient sur l'onglet " Appartenances" et le nouveau groupe est affiché en vert dans la liste.

Pour supprimer le groupe d'un groupe dont il est membre, il faut sélectionner ce dernier dans la liste et cliquer sur " Supprimer".
